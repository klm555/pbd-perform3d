Release Notes#

2024.05.21#

Version 2.6.0#

FEATURE

  • 후처리 그래프를 한글(hwp) 파일로 출력하는 기능 추가.

CHANGED

  • 각각의 지진파에 대해, 모든 위치(ex. 1,5,7,11)의 층간변위비 게이지의 최대값을 구한 후, 모든 지진파들의 평균값을 계산하여 층간변위비로 산출하도록 수정.

CHANGED

  • 그래프 사이즈, 제목 등의 디자인 변경.

CHANGED

  • 그래프 미리보기(Preview) 시, 그래프 제목이 출력되지 않도록 수정.

FIXED

  • 벽체의 압축변형률, 인장변형률 그래프 크기가 다르게 출력되는 오류` 수정.

FIXED

  • Seismic Design Sheets의 입력 여부에 관계없이, 밑면전단력, 층전단력, 층간변위비의 그래프 미리보기(Preview) 기능이 항상 작동되도록 수정.

2024.02.20#

Version 2.5.3#

CHANGED

  • 연결보 Seismic Design Coupling Beam Sheets의 Design_C.Beam 시트에 Boundary 열이 자동으로 입력되도록 수정.

FIXED

  • 층간변위비 그래프 미리보기(Preview)에서, 최대고려지진(MCE) 범례에 CP가 아닌 LS로 표기되는 오류 수정.

2024.02.15#

Version 2.5.2#

CHANGED

  • 그래프 출력 시, 그래프 제목이 출력되지 않도록 수정.

FIXED

  • 층간변위비 그래프 출력 시, 최대고려지진(MCE) 범례에 CP가 아닌 LS로 표기되는 오류 수정.

2024.02.06#

Version 2.5.1#

CHANGED

  • 연결보 Drift Gage 위치(location) 개수 제한 없도록 수정.

CHANGED

  • 연결보 Seismic Design Sheets의 수정에 따라, 중력하중 결과도 자동으로 입력되도록 수정.

2024.01.15#

Version 2.4.2#

FEATURE

  • 연결보 양단고정/1단고정 설정 기능 추가.

FIXED

  • Nodal Loads 값이 모두 Import되지 않는 오류 수정.

2023.09.27#

Version 2.4.1#

CHANGED

  • 후처리 시, Seismic Design Sheets의 입력과 그에 따른 그래프 출력을 구분하여 작동되도록 수정.

2023.09.22#

Version 2.4.0#

FEATURE

  • Seismic Design Sheets에 기존값이 있을 시, 자동으로 삭제 후 입력.

  • 연결보 전단강도 그래프 확인 및 출력 가능.

  • Seismic Design Sheets에서 해석결과가 없는 부재는 Table 및 Plot 시트에서 자동으로 제거.

CHANGED

  • 모든 그래프의 “DE”를 “1.2*DBE”로 변경

FIXED

  • pdf 연속 출력 시, 엑셀 파일이 닫히지 않고 계속 남아서 출력이 점점 느려지는 사항 수정.

  • 입력값의 앞뒤에 공백이 있는 경우, 자동으로 공백 제거.

  • 입력창(checkbox, editbox)이 공란인 경우, 예외 처리.

2023.09.06#

Version 2.3.1#

CHANGED

  • 층 분할 시, 벽체 축변형률(Wall Axial Strain) 게이지가 층 분할을 고려하지 않고 단일 게이지로 Import되도록 수정.

  • 그래프 워드(.docx)로 출력할 때 해상도(dpi=150) 개선.

  • UI 글씨체 및 포맷 변경

2023.08.31#

Version 2.3.0#

FEATURE

  • 프로젝트, 건물명 입력창 추가(pdf 출력용).

  • 메뉴바에 계산 시트 경로로 바로 이동할 수 있는 메뉴 추가.

  • Word(.docx)파일로 출력하는 기능 추가.

  • Word(.docx)파일 출력 시, overwrite되지 않고 새로운 파일이 생성되도록 함.

CHANGED

  • 모든 DCR 그래프의 x축 limit가 3에서 2로 변경됨.

FIXED

  • Nodal Load를 Import할 시, Nodal Load 시트에 비어있는 행이 있는 경우, Error가 발생하는 버그 수정.

  • Analysis Results 파일 선택 시, 아무것도 선택하지 않고 확인을 누르면 기존 경로가 삭제되는 버그 수정.

2023.08.29#

Version 2.2.0#

FEATURE

  • Release Note, 소프트웨어 정보 확인 창 추가.

  • Seismic Design Sheets(부재별 결과확인 시트)에서 벽체 축변형률(Wall Axial Strain) 결과 확인 가능.

  • 연결보 회전각(Beam Rotation) 결과에 Scale Factor 적용 기능 추가.

  • Seismic Design Sheets의 ETC 시트 자동으로 입력.
