==================
Nodes / Elements
==================

Nodes와 Elements 시트에는 Midas Gen에서 Import할 Nodes와 Elements 정보를 입력합니다.

.. topic:: What to do
    
   * Midas Gen에서 레퍼런스 모델을 열고, :guilabel:`Nodes Table`\과 :guilabel:`Elements Table`\의
     데이터를 각각 Nodes, Elements 시트에 :kbd:`Ctrl`\ + :kbd:`C`\, :kbd:`Ctrl`\ + :kbd:`V`\ 합니다.
     
     .. only:: html

        .. figure:: _static/images/DCS_elements.gif
           :align: center

.. warning::

   * 기존에 사용하던 시트에 덮어씌우는 경우, 기존 시트의 데이터가 지워지지 않고 남아있는지 다시 한 번 확인합니다.
   * Nodes 정보를 Import하지 않더라도, Elements, Nodal Loads, Gages, Imbedded Beams 등의 정보를 Import하려면 **Nodes 시트가 반드시 작성되어야 합니다.**
